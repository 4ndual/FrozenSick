## Common Speech-to-Text Substitution Errors

These are NOT name errors (see `session-extraction.mdc` for names). These are common Spanish words that the speech-to-text engine systematically mishears in gaming context.

| Garbled Form | Likely Correct Form | Context Clue |
|-------------|-------------------|--------------|
| country | cantrip | When discussing spell types |
| walkings | warlocks | When discussing enemy casters |
| cristalcalina | cristal / cristalina | Compound word split |
| cotillean / cotilleras | gatillas / gatillo | Mechanism / trigger context |
| gelish reebok | helish rebuke / Hellish Rebuke | Spell casting context |
| snick | sneak (as in Sneak Attack) | Combat context |
| dragón fox | Dragonforce | Organization reference |
| man doble | mandoble (greatsword) | Weapon description |
| necromander | nigromante / necromancer | Undead creation context |
| flechas en la cabeza | headshot | Combat description |
| aguantando | resistiendo | Endurance context |
| pito | whistle / silbato | Equipment context (also crude joke potential — META) |
| intercom | piedrita comunicadora | Communication context |
| gonorreo | furioso / enraged | Colombian slang for angry |
