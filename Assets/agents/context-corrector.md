### Ability Names to Watch For

These are spells and abilities used in combat. The speech-to-text engine often mangles them:

- mano de mago → Mano de Mago (Nixira's spell)
- vociferación → Vicious Mockery (Nixira's combat cantrip)
- eldritch blast → Eldritch Blast (Zacarías's cantrip)
- hellish rebuke → Hellish Rebuke (Zacarías's reaction spell)
- burning hands → Burning Hands (Zacarías's cone of fire)
- sugestión / suggestión → Suggestion (Zacarías's mental manipulation)
- sneak attack / snick attack → Sneak Attack (Tidus's extra damage)
- command / comando → Command (Nixira or Zacarías)
- color spray → Color Spray (Nixira's blinding flash)
- piedrita comunicadora → Communication Stone (cross-distance voice relay)
